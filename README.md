# aluminium

这个项目是一个仿《崩坏: 星穹铁道》的网页/命令行回合制游戏. 目前处于机制开发/测试环节.

This project is a turn-based game imitated "a certain turn-based anime game". this project is in the mechanism
development/testing phase currently.

## 目标

1. 加入全角色/敌对物种 Add all character and enemy
2. 加入测试版角色/敌对物种 Add character and enemy in test version
3. 完善事件系统 Improve event system
4. 开发网页及 API Develop website and API
5. 编写功能测试/单元测试 Write functional tests and unit tests.
